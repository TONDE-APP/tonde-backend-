# Projet Principal : TONDE

TONDE signifie "file d'attente" en Kirundi.

Mission :

Transformer le chaos de l'attente en une expérience claire, digne et prévisible.

TONDE est une plateforme intelligente de gestion des files d'attente destinée aux :

* Banques
* Hôpitaux
* Universités
* Administrations
* Institutions à fort trafic

---

# Vision Produit

* Prendre un ticket à distance

Permettre aux institutions de :

# Queue Engine

Le Queue Engine est le cœur de TONDE.

Fonctions :

* Gestion des tickets
* Priorités
* Temps d'attente estimé
* Synchronisation temps réel
* Notifications

États MVP :

* WAITING
* CALLED
* SERVING
* DONE
* ABSENT
* TRANSFERRED
* CANCELLED
* INCOMPLETE

# Contexte Africain

TONDE est conçu pour fonctionner dans :

* Burundi
* RDC
* Afrique de l'Est

Contraintes :

* Internet instable
* Réseaux 3G
* Faible coût de données
* Matériel limité

Le système doit être :

* Offline First
* Rapide

* Accessible
