# TONDE - Modèle d'Adhésion Utilisateur

Version : MVP v1.0

## Objectif

TONDE est une plateforme multi-organisations.

Un même utilisateur peut appartenir à plusieurs organisations :

* BANCOBU
* CHUK
* KIRA HOSPITAL
* ECOBANK
* Université du Burundi

L'application TONDE est unique.

L'utilisateur n'installe pas une application différente pour chaque institution.

---

## Inscription

Après création du compte TONDE :

* Téléphone
* OTP
* Nom

L'utilisateur doit rejoindre une organisation.

---

## Méthodes d'adhésion

### Niveau 1 : Code d'invitation

L'organisation fournit un code.

Exemple :

BANCOBU-2026

L'utilisateur saisit ce code.

TONDE vérifie le code.

Si valide :

L'utilisateur devient membre de BANCOBU.

---

### Niveau 2 : Numéro Client

L'utilisateur saisit :

* Numéro de compte bancaire
* Numéro dossier patient
* Numéro étudiant

Exemple :

123456789

TONDE vérifie auprès du système de l'organisation.

Si valide :

L'utilisateur rejoint automatiquement l'organisation.

---

## Vue Utilisateur

Lorsqu'il ouvre TONDE :

Mes organisations :

* BANCOBU
* CHUK

L'utilisateur choisit l'organisation avec laquelle il souhaite interagir.

---

## Bénéfices

* Une seule application
* Plusieurs institutions
* Moins de friction
* Croissance rapide de l'écosystème TONDE
